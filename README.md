## fallow

Dead code and duplication analyzer for JS/TS. Finds unused files, exports, types, dependencies, enum members, class members, unresolved imports, and duplicate code across your codebase. 3-40x faster than knip, 16-33x faster than jscpd.

```sh
npx fallow check    # dead code analysis — zero config, sub-second
npx fallow dupes    # duplication detection
```

84 framework plugins, LSP server, VS Code extension, MCP server for AI agents.
