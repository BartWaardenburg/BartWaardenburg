### fallow

Dead code and duplication analyzer for JS/TS. 3-40x faster than knip, 16-33x faster than jscpd.

```sh
npx fallow check    # dead code — zero config, sub-second
npx fallow dupes    # duplication detection
```

84 framework plugins, VS Code extension, LSP, MCP server. [Docs](https://docs.fallow.tools) | [Repository](https://github.com/fallow-rs/fallow)
